# 项目简介

## 核心功能
- 登录注册
- 时事热点
- 新闻推荐
  - 推荐算法：权重衰减、标签推荐、区域推荐、热点推荐
- 用户中心
- 新闻数据管理
- 用户数据管理
- 爬虫系统管理
- 推荐系统管理

## 适用场景
- 新闻资讯平台
- 社交媒体
- 内容聚合网站

## 技术特点
- 使用 Django 框架进行后端开发
- 基于 MySQL 数据库存储数据
- 采用 Vue 框架进行前端开发
- 利用爬虫技术获取新闻数据
- 基于用户行为的协同过滤推荐算法实现个性化推荐
